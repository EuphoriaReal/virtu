# Fonctionnement Inodes

## Compilation

```bash
make
```

## Utilisation

```bash
./inodes_demo big      # Gros fichier 50 Mo
./inodes_demo inode    # 10000 petits fichiers
./inodes_demo reset    # Nettoyage
```

## Nettoyage

```bash
make clean      # Supprime le binaire
make mrproper   # Supprime aussi fs_demo/
```
